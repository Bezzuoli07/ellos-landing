# Teste tecnico - Ellos Design

Landing page desenvolvida para o teste tecnico da Ellos Design, com foco principal em experiencia visual, responsividade e qualidade da interface.

A entrega prioriza a camada de front-end. A parte de backend em PHP e MySQL foi implementada apenas como apoio funcional local para o formulario e nao foi publicada online.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL

## O que foi desenvolvido

- Landing page responsiva
- Hero principal com formulario
- Secoes de acolhimento, beneficios, depoimentos e CTA final
- Identidade visual personalizada
- Validacao basica dos campos no frontend
- Estrutura minima de backend para registro local dos dados do formulario

## Estrutura do projeto

- `index.html`: pagina principal
- `assets/css/styles.css`: estilos e responsividade
- `assets/js/app.js`: animacoes, interacoes e envio do formulario
- `api/submit.php`: endpoint de validacao e envio
- `database.sql`: estrutura da tabela no banco
- `config.example.php`: exemplo de configuracao local

## Observacao sobre o backend

A integracao com PHP e MySQL foi desenvolvida e validada apenas em ambiente local.

Como meu foco principal nesta candidatura e front-end, a versao publicada online representa a interface do projeto. O backend nao foi disponibilizado em ambiente online.

## Observacoes

- O projeto foi pensado para ser simples, organizado e facil de avaliar.

- O backend utiliza `PDO` e `prepared statements` para a insercao local dos dados.
